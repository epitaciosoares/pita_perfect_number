## Sobre

**Pita - Números Perfeitos** é uma aplicação educacional, desenvolvida para fins de estudo e ensino de conceitos matemáticos relacionados aos números perfeitos. A ferramenta oferece uma interface intuitiva para exploração interativa desses números fascinantes.

##  Funcionalidades

- **Verificar Números Perfeitos** - Valide se um número fornecido é perfeito
- **Encontrar Intervalo** - Descubra todos os números perfeitos entre dois valores
- **Interface Intuitiva** - Experiência de usuário clara e amigável
- **Cálculos Eficientes** - Algoritmos otimizados para performance

## O que é um Número Perfeito?

Um **número perfeito** é um número natural que é igual à soma de seus divisores próprios (divisores positivos menores que o próprio número).

### Exemplos:

- **6** = 1 + 2 + 3 (números perfeitos mais antigos conhecidos)
- **28** = 1 + 2 + 4 + 7 + 14
- **496** = 1 + 2 + 4 + 8 + 16 + 31 + 62 + 124 + 248

Os números perfeitos são extremamente raros. Até hoje, apenas 51 números perfeitos foram descobertos!

## Instalação

### Passo 1: Clone o repositório

```bash
git clone https://github.com/seu-usuario/pita-perfect-number.git
cd pita-perfect-number
```

### Passo 2: No diretorio ppn_app Instale as dependências

```bash
flutter pub get
```

### Passo 3: No diretorio ppn_backend Instale as dependências

```bash
flutter pub get
dart run build_runner build 
```

### Passo 4: No diretorio ppn_backend execute a aplicação

```bash
  dart run .\bin\server.dart
```


### Passo 5: No diretorio ppn_app execute a aplicação

só execute os apps na plataforma que preferir.
